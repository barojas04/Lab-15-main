# Referencias Consultadas

Para la realización de la API de DiscoStore se han consultado las siguientes fuentes y documentaciones oficiales:

- [Documentación Oficial de Node.js](https://nodejs.org/docs/latest/api/) - Para el uso de módulos nativos como `fs` y `path`.
- [Documentación Oficial de Express.js](https://expressjs.com/) - Para el enrutamiento, middlewares (`express.json`, `express.static`) y control general del servidor web.
- [NPM: sqlite3](https://www.npmjs.com/package/sqlite3) - Para comprender la conexión y ejecución de consultas en una base de datos SQLite sin usar ORMs.
- [Zod - Documentación Oficial](https://zod.dev/) - Para la construcción de esquemas y validaciones robustas de los payloads en las rutas `POST` y `PUT`.
- Material de clase y presentaciones del curso: Referencias sobre API REST, códigos de estado HTTP obligatorios, uso del patrón controlador/ruta, y buenas prácticas en la estructura del proyecto.
